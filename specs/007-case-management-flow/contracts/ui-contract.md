# UI Contract

## 사례 관리

- `/admin/portfolio` 상단에 `사례 추가` 링크가 `/admin/portfolio/new`로 연결된다.
- 각 목록 행에 `편집` 링크가 `/admin/portfolio/{id}/edit`로 연결된다.
- 대시보드·데스크톱·모바일 관리자 메뉴에는 별도 새 사례 링크가 없다.

## 단계별 업로드

- `작업 전 사진 파일 선택` 입력과 `case-image-before-dropzone` 드롭존을 제공한다.
- `작업 후 사진 파일 선택` 입력과 `case-image-after-dropzone` 드롭존을 제공한다.
- 업로드 완료 미디어의 단계는 드롭존 단계와 같아야 한다.

## 고객 확대보기

- 빠른 상세보기 대표 사진: 대표 미디어 단계에 따른 `작업 전` 또는 `작업 후` 배지.
- 작업 전 슬라이드: 우측 상단 `작업 전` 배지.
- 작업 후 슬라이드: 우측 상단 `작업 후` 배지.
- 다른 단계: 작업 전·후 배지 없음.
