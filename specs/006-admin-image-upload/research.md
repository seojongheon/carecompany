# Research: 관리자 이미지 업로드

## Decision: 공통 업로드 드롭존

사례와 랜딩 이미지에 같은 접근 가능한 드롭존을 사용한다. 클릭 가능한 파일 입력을 유지해 드래그앤드롭을 쓸 수 없는 환경도 지원한다.

## Decision: 권장 비율은 경고

이미지 치수를 읽을 수 있을 때만 4:3 또는 16:9와의 차이를 안내한다. 실제 작업 현장 사진의 다양한 비율을 보존하기 위해 유효한 파일을 막지 않는다.

## Decision: Storage 준비와 활성화를 분리

Storage가 꺼진 동안에는 브라우저 미리보기와 콘텐츠 초안만 만든다. 실제 원격 객체 저장과 공개 URL 기록은 활성화 플래그와 정책 적용 뒤에만 수행한다.
