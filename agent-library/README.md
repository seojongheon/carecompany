# Agent Skill Library

Reusable source documents collected from other local projects. These are reference materials, not automatically active instructions.

## Categories

- `reusable-workflows/`: cross-project planning, implementation, review, and branch workflows.
- `framework-specific/nextjs/`: reuse only for matching Next.js/Vitest projects.
- `project-specific/wordcloud-fastapi-supabase/`: preserve for a similar wordcloud product; do not activate in unrelated projects.
- `communication-styles/`: optional response-tone profiles; activate only when the user requests the style.
- `templates/`: reusable documentation templates.

Global `speckit-*` and `superpowers:*` skills are intentionally not duplicated here. They are runtime-provided dependencies and may be updated centrally.
